Контрольні питання

  1. Що таке CSS і яка його роль у веб-розробці?

    CSS (Cascading Style Sheets) — це мова стилів, що використовується для оформлення HTML-документів. Його роль — визначати зовнішній вигляд елементів веб-сторінки.

  2. Які основні властивості в CSS і як вони задаються?

    Основні властивості включають: `color`, `background-color`, `font-size`, `margin`, `padding`, `border`. Вони задаються у вигляді: `property: value;`

  3. Що таке селектори в CSS? Наведіть приклади різних типів селекторів.

    Селектори — це інструмент для вибору елементів HTML. Приклади:

    • Типовий селектор: `div {}`
    • Класовий селектор: `.class {}`
    • Ідентифікаторний селектор: `#id {}`

  4. Яка різниця між класовим та ідентифікаторним селекторами?

    Клас (`.class`) можна застосовувати до кількох елементів, а ідентифікатор (`#id`) — лише до одного.

  5. Як записати коментарі в CSS?

    Коментарі записуються так: `/* Ваш коментар */`

  6. Як змінити фоновий колір, колір тексту та розмір шрифту для елементу?

    Використовуються властивості:

    • `background-color: value;`
    • `color: value;`
    • `font-size: value;`

  7. Як задати межі (borders), відступи (margins) та внутрішні відступи (padding) для елементів в CSS?

    Межі: `border: width style color;`

    Відступи: `margin: value;`

    Внутрішні відступи: `padding: value;`
